# [Frontend 배포 안내]

## [개요]

이 저장소는 **AWS CloudFront + S3** 조합으로 배포되는 정적 프론트엔드 코드를 포함한다.  
S3에 업로드된 빌드 결과물을 CloudFront 배포가 캐싱하여 전 세계 사용자에게 제공하는 구조이다.  

## [빌드 및 배포 시 유의사항]

- 소스 코드를 수정한 뒤에는 반드시 프로젝트를 **빌드해야 한다**.  
- 빌드 결과물 중 **dist-assets** 디렉터리 내부의 파일들은 해시 기반 파일명으로 생성될 수 있다.  
- 배포 전에 **파일명이 변경되었는지 확인**하고, 필요하다면 관련 참조(HTML, 설정 파일 등)가 정상적으로 최신 파일명을 가리키는지 검증해야 한다.  

## [빌드 환경에 대한 제한 사항]

- 이 프로젝트는 **테스트용으로 구현된 웹 페이지만**을 포함한다.  
- 빌드에 필요한 **Node.js 모듈(node_modules)**이나 기타 개발·빌드용 설정 파일은 제공하지 않는다.  
- 따라서 실제 빌드 작업은 별도의 개발 환경에서 필요한 의존성을 설치한 뒤 진행해야 한다.  

위 내용을 참고하여, 코드 수정 시에는 로컬 또는 별도 CI 환경에서 빌드 후 S3 업로드 및 CloudFront 캐시 무효화(invalidation)를 수행하는 절차를 갖추는 것을 권장한다.
